Welcome

The tamacat2 is an open source Java Web Server / Reverse Proxy software, powered by " Apache HttpComponents".

This is a customizable HTTP/HTTPS Server / Reverse Proxy framework.

Features

  • Standards based, pure Java HTTP Server (Support HTTP/1.1)
  • Pluggable architecture for custom request/response handler and filters
  • Reverse Proxy and Static Contents Web Server
  • Servlet/JSP Integration (Jetty / Tomcat)
  • Jakarta EE MicroProfile Integration (Helidon MP)
  • Required Apache HttpComponents 5.3
  • Required Java Platform, Standard Edition 21+ (JRE/JDK)
